# Code Review Skill
## When to use this skill
- When a pull request needs a formal review
- When the code follows the project's coding standards
## How to use it
1. Analyze the code changes in the pull request
2. Check against the project's coding standards
3. Generate a report in markdown format with sections: 
   - Summary
   - Key changes
   - Potential issues
   - Suggestions
